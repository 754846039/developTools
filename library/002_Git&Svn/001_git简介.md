## git
- 分布式版本控制系统
- 一个人开发项目
  - 把代码回退到之前的某个版本
  - 实现本地代码和远程服务器代码之间的同步

- 团队开发
  - 方便团队合作开发
  - 提高安全性
  - 及时查看进度


## github
- 一个网站
- 基于git的代码托管平台
- 程序员的Facebook
- github网站对应的一个应用程序
- 有大量开源项目
- 方便搜索资源
- 在github上上传一些自己的项目 方便保存
- 通过远程地址访问
- 进行团队协作开发
VCS版本控制系统  历史记录


利用github协作开发
先同步
再commit
再提交
